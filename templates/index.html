Tweet Location Visualization

This app will allow you to search Twitter for the 100 most recent tweets for any keyword or hashtag. It utilizes the tweets location information to plot markers on the map below. Click on any of the markers to see the original tweet.

Out of the 100 Tweets

  • of the tweets have no location data
  • are based on what the user put for location
  • Only tweets were location enabled
  • Basically, this information is pretty inaccurate...
  • But it does give some interesting insight, enjoy!

APIs Used

  • Twitter API
  • Google Maps API
  • Geonames API
